Дополнительные возможности git commit:

git commit -a -m "Message" == git commit add . + git commit -m "Message"

Единственное отличие - Не добавляется в staged зону новые untracked файлы, толькo modified.

----------------------------------------------------------------------------------------------
git commit --amend -m "New or Old Message"

Дополняет последний коммит, добавляя в него свежие изменения.
Также, меняет сообщение последнего коммита.
Новый коммит не создаётся
