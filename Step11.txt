git checkout - используется для перемещения между коммитами, версиями отдельных файлов и ветками.
После перехода указатель HEAD находится в состоянии DETACHED. Любые изменения будут удалены.

git checkout hash_code
git checkout master    - Возвращает на ветку master и на последний коммит
-------------------------------------------------------------------------
git checkout -- File2.txt         - удаляет modified: File2.txt
-------------------------------------------------------------------------
git checkout hash_code -- File2.txt         - будет изменён только File2.txt
